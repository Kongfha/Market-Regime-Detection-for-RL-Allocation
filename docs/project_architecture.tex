\section{Research Objective}

\textbf{Research question:} Can latent market regimes discovered from weekly price,
volatility, cross-asset, and macroeconomic features improve ETF allocation performance
compared with price-only decision-making?

The project proceeds in two phases: (1)~pattern recognition to discover market regimes
from historical features, then (2)~reinforcement learning to map regime context into
portfolio weights across three tradable ETFs (SPY, TLT, GLD) plus cash.

\subsection{Current Focus}

The current phase focuses exclusively on \textbf{pattern recognition} using three core
weekly feature tables: market features, macroeconomic features, and forward asset
targets. The news sentiment module has been archived (see Section~\ref{sec:legacy}).

\textbf{Causal lag rules:}
\begin{itemize}[nosep]
    \item Daily FRED series (DFF, DGS3MO, DGS10, T10Y3M, T10YIE, BAMLH0A0HYM2, VIXCLS, DTWEXBGS): 0 lag --- use last value in the decision week

\end{itemize}

\section{Data Pipeline Architecture}

\subsection{Pipeline Flow}

\begin{enumerate}[nosep]
    \item \textbf{Fetch raw data:} Yahoo Finance daily OHLCV $\rightarrow$ \texttt{data/raw/yahoo\_prices\_daily.csv}
    \item \textbf{Fetch macro data:} FRED API (core 16 series) $\rightarrow$ \texttt{data/raw/fred\_macro\_panel.csv}
    \item \textbf{Build daily features:} 28 technical features from OHLCV $\rightarrow$ \texttt{data/processed/market\_features\_daily.csv}
    \item \textbf{Aggregate to weekly:} W-FRI calendar $\rightarrow$ \texttt{data/processed/market\_features\_weekly.csv}
    \item \textbf{Process macro:} Pivot, transform, lag-align $\rightarrow$ \texttt{data/processed/macro\_features\_weekly.csv}
    \item \textbf{Compute targets:} Forward returns $\rightarrow$ \texttt{data/processed/weekly\_asset\_targets.csv}
    \item \textbf{Join:} Inner join on \texttt{week\_end} $\rightarrow$ \texttt{data/processed/model\_state\_weekly\_price\_macro.csv}
\end{enumerate}

\section{Legacy: News Module}
\label{sec:legacy}

The news sentiment module has been archived in \texttt{legacy/} because Yahoo
Finance's news API only returns recent headlines (last few weeks), making it
unsuitable for building historical sentiment features covering the full 2014--present
training window. An alternative news data source is being evaluated.

See \texttt{legacy/README.md} for the archived data inventory and instructions
for reviving the module when a replacement source is identified.
