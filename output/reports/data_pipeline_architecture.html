→
📅 Market Features (Weekly)
Resampled to W-FRI calendar
~30 features aligned to decision dates
Includes week_end and week_last_trade_date
market_features_weekly.csv ~484 KB
⚙ Macro Features (Weekly) — Lag-Aware Causal Alignment
~50 features from FRED panel with frequency-aware lags:
Daily series (DFF, DGS10, etc.) → use last value in week, no lag
Weekly series (NFCI, ICSA, etc.) → lag by 1 week before joining
Monthly series (CPI, UNRATE, etc.) → lag by 1 month before joining
Each series applies its suggested_transform: level, change, pct_change, log, etc.
macro_features_weekly.csv ~181 KB Causal
⚙ News Events Enriched

Sentiment (pos/neg/neutral) • Topic classification

🎯 Weekly Asset Targets (RL Reward Signal)
Next-period returns for SPY, TLT, GLD shifted forward by 1 week

Detailed Feature Mapping: Source → Feature → Module

📈 Price → Market Features

  • Returns block (9 features): {spy,tlt,gld}_ret_{1d,5d,20d}
  • Volatility block (6): {spy,tlt,gld}_vol_{5d,20d}
  • Drawdown (3): {spy,tlt,gld}_drawdown_60d
  • MA Gap (3): {spy,tlt,gld}_ma_gap_5_20
  • Volume (3): {spy,tlt,gld}_volume_z_20
  • Intraday Range (3): {spy,tlt,gld}_intraday_range
  • Cross-asset (4): qqq_spy ratios, correlations
  • Context (4): vix_level/chg, tnx_level/chg

🏛 FRED → Macro Features

  • Policy & Curve: DFF, DGS3MO, DGS10, T10Y3M
  • Inflation & Risk: T10YIE, BAMLH0A0HYM2, VIXCLS, DTWEXBGS
  • Liquidity & Labor: WRESBAL, ICSA, UNRATE
  • Activity & Demand: CPIAUCSL, CFNAI, UMCSENT, PERMIT
  • Optional: DGS30 (via core_plus_duration preset)
  • Each series → level + transform (change/pct/log)

📰 News → News Features

  • Sentiment: mean_sentiment_score, neg/pos ratio
  • Volume: headline_count, relevant_headline_count
  • Impact: mean_impact_score, max_impact_score
  • Topics (5 ratios): growth_earnings, inflation_rates, recession_slowdown, geopolitics_policy, market_technical_flow
  • Relevance: relevance_ratio

🎯 Targets → RL Reward

  • next_return_spy: next week SPY return
  • next_return_tlt: next week TLT return
  • next_return_gld: next week GLD return
  • Shifted forward 1 week (no look-ahead)
  • Used as reward signal for RL agent
  • Also used for regime quality evaluation

Causal Alignment Rules

⏰ Decision Calendar

  • Weekly frequency: W-FRI
  • Each row = one decision week
  • Features use only past data available at decision time
  • Targets are next-week returns (shifted forward)

🔒 Release Lag Rules

  • Daily FRED: 0 lag (same-week value)
  • Weekly FRED: 1-week lag (prior week release)
  • Monthly FRED: 1-month lag (prior month release)
  • News: published_at → assign to week_end period
